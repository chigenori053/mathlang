# Pro Directory Overview

- `api/` — programmatic helpers (`client.py`, `examples.py`) used by researchers.
- `tools/` — visualization and inspection utilities for causal graphs/logs.
- `notebooks/` — placeholder for R&D notebooks (to be added incrementally).
- `demo/` — scenario runner/readme for causal/counterfactual CLI demos.
- `config/` — research defaults (settings/logging).
- `examples/` — DSL v2.5 programs targeted at pro workflows.
